# Vision-Prompt
A Streamlit web app that accepts text prompts and images as input and generates AI-powered responses using Google Gemini 1.5 Pro.
## Tech Stack
- Python
- Streamlit
- Google Gemini 1.5 Pro API
- Pillow (PIL)
- dotenv

## Features
- Text prompt input
- Image upload support (PNG, JPEG, JPG, WEBP)
- AI-generated responses using Gemini

## How to Run
1. Clone the repository
2. Install dependencies: pip install -r requirements.txt
3. Add your GOOGLE_API_KEY in .env file
4. Run: streamlit run vision.py
